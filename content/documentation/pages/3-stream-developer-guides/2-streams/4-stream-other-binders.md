# Introduction

Spring Cloud Stream provides binders for the following messaging middleware and streaming platforms:

- RabbitMQ
- Apache Kafka
- Amazon Kinesis
- Google Cloud PubSub
- Solace's PubSub+
- Azure Event Hubs
